# Monsters Shattered Pixel Dungeon---怪物的像素地牢

**怪物的像素地牢**是一个基于**破碎的像素地牢**的个人模组,从整体上而言难度会比原版会**简单**一些。

所有代码都基于**破碎的像素地牢**,在这里感谢其作者**Evan**!
